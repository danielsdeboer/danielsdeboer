How I Learned To Stop Worrying And Love The CSS

I'll be honest with you. For a while I didn't really get CSS. Everything I tried to do was super-hard and stuff didn't do what I wanted to and things like position: absolute were some kind of inscrutable black magic (I know, I know)...

Until one of my developer friends passed me a draft copy of a book called CSS In Action. Suddenly it all clicked. Positions, inheritance, specificity, pseudo-classes, displays, media queries all made sense.

But that's history. Now I use CSS the way it was intended (and a few ways it wasn't), all to make pages that look great everywhere. I don't know if I'm exactly a CSS guru, but I'm at least a CSS sensei.
